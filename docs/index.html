MusicXML is a music interchange format designed for music notation, analysis, retrieval, and performance applications. The MusicXML format has been initialy developped by Recordare. It is open for use by anyone under a royalty-free license.

libMusicXml is a portable library designed close to the MusicXML format and intended to facilitate MusicXML support. It's an open source project hosted on github that includes notably high level tools for conversion to other music notation formats like Guido Music Notation format, Lilypond and Braille Music Notation.
