\abstract {
This document shows the architecture of the \mf\ library, to be found at
\url {https://github.com/jacques-menu/musicformats}.

\mf\ is written in \CPlusplus\ and provides a set of music scores formats and converters between various textual music scores formats. Building it only requires a \CPlusplus\ compiler and \cmake.
